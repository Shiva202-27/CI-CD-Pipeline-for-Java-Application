# CI/CD Pipeline for Java Application

This project demonstrates a complete CI/CD pipeline using Jenkins, Docker, and AWS EC2.

## Tools Used
- GitHub
- Jenkins
- Docker
- AWS EC2
- Nginx

## Project Flow

GitHub → Jenkins → Docker Build → Deploy Container on AWS EC2

## Features

- Automated build using Jenkins Pipeline
- Dockerized web application
- Deployment on AWS EC2 instance
- CI/CD workflow from GitHub

## Access Application

http://<EC2-IP>:8082
